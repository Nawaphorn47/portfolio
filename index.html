Nexus-API: สถาปัตยกรรม Backend แบบ Full-Stack ที่ยืดหยุ่นและพร้อมใช้งานจริง

Tech Stack: Node.js, TypeScript, Express.js, PostgreSQL, Docker, Docker Compose, Nginx, JWT, Prisma ORM, bcryptjs

ออกแบบและพัฒนาระบบ RESTful API ที่พร้อมสำหรับการนำไปใช้งานจริง (Production-ready) โดยใช้เทคโนโลยีสมัยใหม่ เน้นการจัดการโครงสร้างที่ดูแลรักษาง่าย และจำลองสภาพแวดล้อมการทำงานที่มีความเสถียรสูง

  • Layered Architecture: นำสถาปัตยกรรมแบบแยกส่วนมาใช้จัดการโครงสร้างโปรเจกต์ เพื่อให้โค้ดมีความเป็นระเบียบ ทำงานร่วมกันได้ง่าย และสะดวกต่อการดูแลรักษา (Maintainability)
  • Security & Authentication: พัฒนาระบบยืนยันตัวตนผู้ใช้ด้วย JWT (JSON Web Token) ทำงานร่วมกับการแฮชรหัสผ่านด้วย bcryptjs เพื่อความปลอดภัยสูงสุด
  • Database & ORM: จัดการฐานข้อมูล PostgreSQL ผ่าน Prisma ORM ช่วยให้การดึงและจัดการข้อมูลมีประสิทธิภาพและมีความปลอดภัยระดับ Type-safe (TypeScript)
  • Infrastructure & Containerization: จัดการระบบทั้งหมดให้อยู่ในรูปแบบ Container ด้วย Docker และ Docker Compose (รวมถึง Database และ Nginx) ช่วยแก้ปัญหาความเข้ากันได้ของระบบปฏิบัติการ และทำให้การ Deploy เป็นเรื่องง่าย

Queue Management System (ระบบ API จัดการคิว)

Tech Stack: Node.js, Express.js, PostgreSQL, Docker, Docker Compose, Nginx

ออกแบบและพัฒนาระบบ REST API สำหรับจัดการคิวการให้บริการแบบครบวงจร รองรับฟังก์ชันการจองคิวใหม่, การเรียกคิวถัดไปแยกตามช่องบริการ (Counter), และการดึงข้อมูลสถานะคิว (Active/Waiting) เพื่อนำไปแสดงผลบนหน้าจอ Display

  • Database & Logic: ออกแบบและจัดการฐานข้อมูลด้วย PostgreSQL โดยมีการเขียนสคริปต์สร้างตารางอัตโนมัติ (Auto-migration) เมื่อรันระบบ และเขียน SQL Query จัดการสถานะคิว
  • Infrastructure: จัดการสภาพแวดล้อมของระบบทั้งหมด (API, Database, Web Server) ให้อยู่ในรูปแบบ Container ด้วย Docker และ Docker Compose
  • Networking: ตั้งค่า Nginx ให้ทำงานเป็น Reverse Proxy สำหรับจัดการ Traffic ที่วิ่งเข้าสู่ API และเปิดใช้งาน CORS เพื่อให้ฝั่ง Client สามารถดึงข้อมูลไปแสดงผลได้อย่างไร้รอยต่อ

Smart Farm Data Management System (ระบบจัดการข้อมูลฟาร์มอัจฉริยะ)

Tech Stack: Node.js, Express.js, InfluxDB, Grafana, Docker

พัฒนาระบบ Backend และ Infrastructure สำหรับรับส่ง จัดเก็บ และวิเคราะห์ข้อมูลจากเซนเซอร์ IoT (Sensor Data) ภายในฟาร์ม เพื่อรองรับการแสดงผลสถานะแบบ Real-time และการจัดการข้อมูลจำนวนมากอย่างมีประสิทธิภาพ

  • Time-Series Database: นำ InfluxDB มาใช้ในการจัดเก็บข้อมูลจากเซนเซอร์โดยเฉพาะ เพื่อรองรับการเขียนข้อมูล (Write-heavy) ปริมาณมากอย่างรวดเร็วและประหยัดพื้นที่
  • Data Visualization & Monitoring: เชื่อมต่อฐานข้อมูลเข้ากับ Grafana เพื่อสร้าง Dashboard สรุปผลข้อมูลสภาพแวดล้อม (เช่น อุณหภูมิ, ความชื้น) ให้มอนิเตอร์และวิเคราะห์ย้อนหลังได้
  • Backend & API: ออกแบบ API ด้วย Node.js เพื่อจัดการข้อมูลและประมวลผลการทำงานอัตโนมัติต่างๆโดยความยากคือเรามีการสื่อสารผ่าน mqtt protocal ด้วย เพื่อตอบสนองการเปลี่ยนแปลงของสภาพแวดล้อมในฟาร์ม
  • Infrastructure: จัดการสภาพแวดล้อมการทำงานของระบบ (Database, API, Dashboard) ให้อยู่ในรูปแบบ Container ด้วย Docker เพื่อความง่ายต่อการติดตั้งและนำไปใช้งานจริง
